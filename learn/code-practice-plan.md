# Code Practice Plan

## Purpose

This file tells the student what to build with her hands.

Learning becomes stronger when she writes or reads code with a clear goal.

## Day-Wise Practice

### Day 1

- Write the full project flow in 5 sentences.

### Day 2

- Read `AppConfig.java` and list every config value.

### Day 3

- Read `uploadApi.ts` and explain every async step.

### Day 4

- Draw the full architecture from memory.

### Day 5

- Read `openapi.yaml` and write the request and response fields.

### Day 6

- Explain where Cognito fits and where it does not fit.

### Day 7

- Explain the pre-signed URL flow step by step.

### Day 8

- Explain what the image processor reads and what it writes.

### Day 9

- Walk through one request from browser to backend to storage.

### Day 10

- Read `UploadUrlHandler.java` and identify handler, validation, and service layers.

### Day 11

- Read `ImageProcessingService.java` and identify input, transformation, and output.

### Day 12

- Read `docker-compose.local.yml` and explain each service.

### Day 13

- Read `infra/template.yaml` and map local services to production AWS services.

### Day 14

- Write a debugging checklist for missing final image.

### Day 15

- Explain the full project in 1 minute and 5 minutes.
