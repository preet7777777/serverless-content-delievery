# Capstone Rubric

## Purpose

Use this file at the end of the 15-day learning path.

It helps the teacher judge whether the student is:

- only remembering words
- actually understanding the system
- ready for intern-level interviews

## Scoring Method

Score each section from `1 to 5`.

- `1` = very weak
- `2` = partial understanding
- `3` = acceptable beginner level
- `4` = strong beginner level
- `5` = interview-ready beginner level
